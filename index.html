What information we collect

  • Retrieve running apps: This permission is used in Boost in order to view the running apps.
  • Read the contents of your USB storage: This permission allows to view app data(image, text, audio etc..) stored on their SD card.
  • Modify or delete the contents of your USB storage: This permission allows to delete app data(image, text, audio etc..) stored on their SD card.
  • View Wi-Fi connections: This permission is used in settings and notification toolbar, in order to view Wi-Fi connections.
  • Manage document storage: This permission is used for read/writing File Commander files to and from your device.
  • Update component usage statistics: This permission allows the app to modify collected component usage statistics. Not for use by normal apps.
  • View network connections: This permission is used in settings and notification toolbar, in order to view network connections.
  • Full network access: This permission is used to access the device’s network for certain functions including receiving update notifications or accessing app classification labels.
  • Run at startup: This permission is used for file search in the File Commander file browser
  • Draw over other apps: This permission allows the Free Swipe tab to be displayed as a floating window on other apps.
  • Control vibration: This permission allows to make the phone vibrate once after users set the device on vibrate, confirming the setting has been turned on.
  • Bind device admin: This permission ensure that only the system can interact with the receiver (no application can be granted this permission). This prevents other applications from abusing your device admin app. .
  • Email address: App takes user email to send reset pass code.
